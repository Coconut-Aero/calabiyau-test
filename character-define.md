剪刀手是反抗欧泊统治的革命组织，目标是建立一个没有压迫的公正世界。

| 角色 | 简介 |
|------|------|
| **明** | 行动队突击队长，从普雷顿街头走出来的姐头 |
| **拉薇** | 前欧泊搜查官，现剪刀手战士，擅用脉冲刀 |
| **梅瑞狄斯** | 剪刀手的核心人物和领导者，前欧泊科研人员 |
| **香奈美** | 高人气的偶像歌手，剪刀手的地下志愿者 |
| **令** | 叛离欧泊的前搜查官，信的兄长 |
| **艾卡** | 想要拯救世界的热血女高中生，佩戴面具的神秘英雄 |
| **珐格兰丝** | 剪刀手的研究员，植物学专家 |
| **玛拉** | 剪刀手最无畏的决斗者，极具战斗天赋 |

| **白墨** | 现任盟主，纨绔不羁，崇尚自由 |
| **玛德蕾娜** | 效力于乌尔比诺的绘师，天真烂漫的少女 |
